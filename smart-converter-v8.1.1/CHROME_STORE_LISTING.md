# Smart Converter — Chrome Web Store Listing

## Short description
Instantly convert currencies and units by highlighting values on any webpage.

## Detailed description
Smart Converter lets you highlight any value on a webpage and instantly see a clean conversion popup.

Convert currencies such as USD, EUR, GBP, RUB, AED, HKD, QAR, PLN, JPY, CNY and many more.

Convert common units between Metric and Imperial, including lbs/kg, ft/in/cm/m, miles/km, gallons/liters, and °F/°C.

Perfect for online shopping, travel planning, reading international content, fitness and nutrition values, finance, crypto, and everyday browsing.

### Features
- Instant popup on highlight
- Light and dark tooltip themes
- Multiple target currencies
- Metric / Imperial mode
- Fast, lightweight, and minimal
- Preferences saved locally

## Privacy note
Smart Converter stores user preferences locally using Chrome Storage. It does not collect personal data, browsing history, or sell user data.
